- **Designed the architecture and data flow** for the automated agentic AI content generation system
- **Dual-model strategy** — integrating AWS Bedrock alongside a free-tier service to balance performance and cost
- **Serverless infrastructure** — since the AI runs once daily, serverless significantly reduces operational costs
- **Monitoring dashboard** planned to track token usage and end-to-end pipeline execution time
- **Free model integration** not yet scoped — AWS Bedrock will be used in the interim
